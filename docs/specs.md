# 仕様

## 概要
`agent-port` は、CLI から挨拶メッセージを表示する Python アプリケーションです。

## 現在の振る舞い
- `build_greeting()` は既定値として `agent-port` を使い、`Hello from agent-port!` を返します。
- `main()` は `build_greeting()` の戻り値を標準出力へ表示します。

## 入出力
- 入力: 現時点ではコマンドライン引数や外部入力を受け取りません。
- 出力: 標準出力へ 1 行の挨拶メッセージを出力します。

## 制約
- 実行環境は Python 3.12 以上を想定します。
- 開発、依存解決、テスト実行には `uv` を使用します。
- 関数、クラス、メソッド、テストには日本語の NumPy 形式 docstring を付けます。

## 更新ルール
振る舞い、責務、入出力、制約が変わる場合は、このファイルを更新してください。パスを記載するときは、リポジトリ基準の相対パスを使ってください。仕様が大きくなった場合は `docs/specs/` 配下へ分割してください。
