# 使い方

## セットアップ
開発環境は `uv` で作成します。

```powershell
uv sync --dev
```

## 実行
アプリケーションは次のコマンドで起動します。

```powershell
uv run python main.py
```

現時点では、標準出力に挨拶メッセージを表示します。

## テスト
自動テストは `pytest` で実行します。

```powershell
uv run pytest
```
